# Data Structure
* 資料結構是可以有效改善記憶體使用效率的方法之一
* 有以下幾種排序方式:
   * 列表(List)
   * 陣列(Array)
   * 堆疊(Stack)
   * 佇列(queue)
   * 雜湊表(Hash Table)
   * 堆積(Heap)
   * 二元搜尋數(binary search tree)
## List
* 分為有序跟無序
* 利用線性搜尋的方式進行
* Big O = O(n)
## Array
* 是一種儲存資料的方式
* 相較於列表更容易存取
* 但新增或刪除數據時較麻煩
* 與列表相同之處:
   * 兩者的資料結構是可變的
   * 可以切片或迭代
   * 都有索引
   * 可用於儲存資料
